# UNPLAN Village Hakuba — Guest Guide

UNPLAN Village Hakuba（白馬・栂池高原）の宿泊者向け現地ガイドサイト（静的サイト）。

## ページ構成
- `index.html` … トップ（日本語 / 英語トグル）
- `stay.html` / `stay-en.html` … 宿泊のご案内
- `hakuba.html` / `hakuba-en.html` … 白馬ガイド
- `green-season.html` / `green-season-en.html` … グリーンシーズン特集
- `images/` … サイト内画像

## 技術メモ
- 素のHTML（CSS / JavaScript はHTML内に直書き）
- フォント: Google Fonts（Outfit / Noto Sans JP）
- 地図: Leaflet（CDN）

## 編集方法
各HTMLを直接編集して `main` ブランチに push すると、GitHub Pages で自動公開されます。
**日本語版・英語版は別ファイル**のため、内容を変更するときは両方を更新してください。
